Very simple game of Rock Paper Scissors I made a month into learning how to program.

It's late because I only recently learned about Github.

Few notes:

1. Allows for single letter inputs for Rock, Paper, or Scissors
2. Can re-roll if both player and computer choose the same input
3. Features input validation! The program won't end or wimp out if the user makes an invalid input

Two very primitive, prehistoric things but I'm proud I thought of that this early
